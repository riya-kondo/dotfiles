## overview
- 自分用の隠しファイルたちです。ホームディレクトリでcloneして使います。
## requirements
- zsh  
## Usage
>$ cd  
>$ git clone https://github.com/riya-kondo/dotfiles.git  
>$ cd dotfiles  
>$ ./setup.sh  
## proxy
- ~/.proxy_setupに追記してproxyを設定できます。
